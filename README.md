# 개발 블로그 플랫폼 YEBLOG

## 프로젝트 개요

이 프로젝트는 CRUD 기본 기능을 학습하고, 난이도가 높은 기능을 추가하여 개발 역량을 향상시키기 위한 목적으로 시작되었습니다. 개발자로서의 역량 향상과 동시에 개발한 블로그 플랫폼을 통해 지식을 공유하고 사용자들을 유입시키는 것을 목표로 합니다.

## 주요 기능
- [X] 회원가입
- [X] 로그인
- [X] 게시물 작성
- [X] 게시물 수정
- [X] 게시물 삭제
- [X] 게시물 검색
- [X] SNS 공유하기
- [X] 회원정보 수정(비밀번호,이메일)
- [X] 마이페이지 사진 업로드

 ### 기능 추가 예정:
- [ ] 댓글 작성 ,수정 , 삭제
- [ ] 알림 기능
